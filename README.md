# HooperMedia
HooperMedia code exercise. 

- An Angular -  client application
- An ASP.NET Core Web API backend
- SQL Server database (via Docker Compose)


## Detailed Documentation

- Client guide: [Client/hooper-media-client/README.md](Client/hooper-media-client/README.md)
- Server guide: [Server/HooperMedia/README.md](Server/HooperMedia/README.md)

## Prerequisites

- Docker Desktop (with Docker Compose)
- Node.js and npm (for local non-Docker client development)
- .NET SDK (for local non-Docker server development)

## Quick Start (Docker)

Start backend services first, then the client. For more details, please verify the README.md (server and client)
